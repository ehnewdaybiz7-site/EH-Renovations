# EH Renovations

A premium, responsive portfolio website for **EH Renovations**, featuring the high-quality home remodeling, carpentry, and total property restoration work of Ernest Harston. 

## Features
- **Filterable Gallery**: Showcase projects categorized by kitchens, bathrooms, exterior framing, interior remodeling, and finish carpentry.
- **Lightbox Viewer**: Interactive zoom-in detail modal for all high-resolution project images.
- **Service Inquiry Form**: Styled contact section with custom validation for prospective clients.
- **Fully Responsive Layout**: Premium CSS Grid and Flexbox layouts featuring a glassmorphic blurred navbar and smooth transition animations.

## Tech Stack
- HTML5
- CSS3 (Vanilla)
- JavaScript (Vanilla)
- Google Fonts (Outfit & Inter)
- FontAwesome Icons

## Hosting on GitHub Pages
This site is hosted using GitHub Pages. The files are organized with `index.html` at the root directory and project images located in the `Renovation Booklet/` folder.
